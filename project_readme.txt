BookMovieApp is a React frontend project helps users in
booking movie tickets.

This repository contains the added code and screenshots of the various
screens of the App.

Executing the project
=====================
1) Start the backend movieapp-master following the appropriate steps
2) Run the BookMovieApp by issuing the commans "npm i" and "npm start" 
   from the BookMovieApp home directory.

If by chance the npm command fail due to lack of the appropriate module,
please install those modules using npm

Tools used in the project
=========================
Apache Maven 3.8.4 
Java version: 11.0.12
OS name: "windows 10", version: "10.0", arch: "amd64", family: "windows"
Node/NPM
